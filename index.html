🎓 MSc Thesis Project — Most Advanced

Cardiac Arrest Prediction using TAN-LSTM

My MSc thesis at TSI University. Predicts intraoperative cardiac arrest at 30, 60, 120 and 240 minutes before the event using VitalDB clinical time-series data (6,388 surgical cases). Built a Temporal Attention Network + LSTM model targeting AUROC > 0.91, handling 90:1 class imbalance with SMOTE-ENN.
